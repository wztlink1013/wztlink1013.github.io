    Navicat-1045报错

    作者: 尼采般地抒情 时间：2020-10-17 01:12:59 分类： Bug 标签： 文章访问量：loading 文章评论数：loading

    一、报错信息

    1045-Access denied for user root@localhost(using password:YES)

    二、问题分析与解决

    不被接入，改 root 密码

    在 mysql 控制台里面改密码

    ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '你的密码';

    刷新数据库

    flush privileges;

    退出

    quit;

    重启 Navicat 再次链接即可。
