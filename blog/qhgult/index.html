art-template 和 express-art-template

为了是art-template模板引擎能够更好地和Express框架配合，模板引擎官方在源art-template模板引擎的基础上封装了express-art-template，所以在使用Espress框架时候，这两个模块都要下载。
  • art-template 官网：https://aui.github.io/art-template/zh-cn/docs/
  • 关于模板引擎：6 个常见 JS 模板引擎

使用模板渲染情况下声明

// 渲染后缀为art的模板时，所使用的模板引擎是什么
app.engine("art", require("express-art-template"));
// 告诉express框架模板的位置
app.set("views", path.join(__dirname, "views"));
// 告诉express框架模板的默认后缀是什么
app.set("view engine", "art");

关于绝对/相对路径的引用

模板资源里面引用静态资源的路径要用绝对路径。怎么使用绝对路径呢，类似 hexo 静态博客一样，前面加一个“/”即可是根目录路径下，再进行根目录路径下路径写法
