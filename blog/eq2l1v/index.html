数据结构+算法分析与设计两门课程的学习笔记

一、数据结构

概念

线结构

  • 顺序表、单链表（单链表最为基础，最为重要）

  • 栈和队列

    操作受限的链表

  • 串

    内容受限的线性表

  • 数组

    多维数组求地址

  • 广义表

    概念，和首位

​

树结构

  • 二叉树的五条性质烂熟于胸，性质 3 的证明
  • 遍历二叉树：前序、中序、后序
  • 线索二叉树
  • 树都可以转换为二叉树：左儿子右兄弟法
  • 森林转换为二叉树：森林中的所有树变成二叉树，然后将其各个根节点变成左右兄弟类型
  • 哈夫曼树，哈夫曼编码：外排序用到

​

图结构

  • 邻接矩阵、邻接表（有向图、无向图）、逆邻接表

    十字链表、邻接多重链表（不要求）

  • 图的遍历：BFS（主要）、DFS（主要）、层次遍历（一层一层遍历）

  • 连通性问题：最小生成树，为了解决 n 个顶点连接起来（工程应用）

    • Prim 算法（稠密图用）
    • Kruskal 算法（稀疏图用）

  • 最短路径问题

    • 算法 1：一点到其他个各点：Dijaskal 算法
    • 算法 2：每一对顶点之间：Floyd 算法

  • 关键路径问题

    • AOV 网：拓扑排序
    • AOE 网：关键路径（工程问题）

查找

  • 静态查找
    • 顺序
    • 折半
    • 分块
  • 动态查找
    • 二叉排序树：构造
    • AVL（平衡二叉排序树）
  • B-树
    • B+树，一个结点放多关键字
  • 哈希（Hash）表及其查找：解决冲突的办法

例子：Euclid-GCD 问题

问题描述
将每个整数分解为素因子的积，找出公共的素因子，它们的积即是 GCD
问题思路

代码实现

package com.wztlink1013.al.EuclidGCD;
import java.util.Scanner;

public class Main {
    public static void main(String args[]){
        Scanner input = new Scanner(System.in);

        System.out.println("请输入两个大于零的自然数：");
        int a = input.nextInt();
        int b = input.nextInt();
        GCD(a,b);
        System.out.println(a + "和" + b + "两个数的GCD值为：" + GCD(a,b));
    }
    public static int GCD(int i, int j){
        int r;
        while (j != 0){
            r = i%j;
            i = j;
            j = r;
            System.out.println("a="+i+"；b="+j+"；r="+r);
        }
        return i;
    }
}
