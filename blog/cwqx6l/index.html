配置java开发环境

作者: 尼采般地抒情 2020-01-27 11:45:16 本文发布时间为2020年01月27日11时45分16秒 分类: Environment 标签: 下面是正文内容 评论数: 阅读量: loading

原文地址：https://www.wztlink1013.com/blog/cwqx6l/，参与评论

一、下载

https://www.oracle.com/cn/java/technologies/javase-downloads.html

二、环境变量配置

在系统变量下

CLASSPATH

.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;%CATALINA_HOME%\common\lib\servlet-api.jar;

其中%CATALINA_HOME%\common\lib\servlet-api.jar;为 tomcat 的加包配置

JAVA_HOME

C:\Program Files\Java\jdk1.8.0_181

path

%JAVA_HOME%\bin
%JAVA_HOME%\jre\bin

三、测试

CMD 下输入java -version
