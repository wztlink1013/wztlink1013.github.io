npm发布第一个包

作者: 尼采般地抒情 时间：2020-02-08 09:21:00 分类： Web 标签： 文章访问量：loading 文章评论数：loading

初始化

在https://www.npmjs.com/注册的用户账号
新建一个文件夹npm-wzutao再次文件夹下进入 bash

登录 npmjs
npm login

初始化
npm init

然后在该文件夹下新建 index.js 文件，写入如下测试代码

exports.sum = function () {
  var res = 0;

  for (var i = 0; i < arguments.length; i++) {
    res += arguments[i];
  }

  return res;
};

发布

npm publish

去官网可以搜索到

更新

【1】本地更新版本号

npm version patch
1.0.1 版本，注意，是最后一位修改了增 1
npm version minor
1.1.0 版本，注意，是第二位修改了增 1
npm version major
2.0.0 版本，注意，是第一位修改了增 1

【2】修改远端的版本
提交到远端 npm 中
npm publish
