MongoDB数据库学习笔记

尼采般地抒情 2021-04-11 17:05:12 Node.js loading 次访问 loading

前言：对 Node.js 支持较好，逻辑和 MySQL 一样。

MongoDB 和 MongoDB Compass

  • MongoDB 中文手册|官方文档中文版
  • MongoDB 教程 | 菜鸟教程

MongoDB 下载安装

  • 下载地址：https://www.mongodb.com/try/download/community



不要勾选，不然下载耗时
最后将安装的 bin 目录路径添加到环境变量中即可

MongoDB Compass 下载安装

（可视化图形软件）

  • 下载地址：https://www.mongodb.com/try/download/compass

安装 BUG：不能开启 mongoDB

原因：权限不足
用 win+X 点击

然后在输入 net start mongoDB 即可

MongoDB 相关使用

MongoDB 数据库命令

  • 查看所有数据库：show dbs
  • 使用指定数据库：use testmongodb

如果有该数据库则切换到该数据库，如果没有则新建再切换至该数据库

  • 展示数据库所有用户：show users
