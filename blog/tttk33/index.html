Git push出(10054)/(port 443：time out)错

作者: 尼采般地抒情 时间：2021-03-23 17:24:46 分类： Bug 文章访问量：loading 文章评论数：loading

报错信息

Git push出(10054)/(port 443：time out)错
git push error: RPC failed; curl 56 OpenSSL SSL_read: SSL_ERROR_SYSCALL, errno 10054

原因

由于 Http 协议错误，当 pull 或者 clone 的时候，或者是 github 某个 CDN 被伟大的墙屏蔽所致。

解决办法 1

//10054复制下面的
git config http.postBuffer 524288000

//443复制下面的
git config --global --unset http.proxy

git config --global --unset https.proxy

解决办法 2

在https://www.ipaddress.com/里面查找下面三个域名的 ip 地址

  • github.com
  • github.global.ssl.fastly.net
  • codeload.github.com

将其添加到本电脑 hosts 文件当中
C:\Windows\System32\drivers\etc\hosts 

CMD 窗口刷新 DNS 缓存
ipconfig /flushdns
